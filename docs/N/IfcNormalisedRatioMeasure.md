IfcNormalisedRatioMeasure
=========================
_IfcNormalisedRatioMeasure_ is a dimensionless measure to express ratio values
ranging from 0.0 to 1.0.  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
