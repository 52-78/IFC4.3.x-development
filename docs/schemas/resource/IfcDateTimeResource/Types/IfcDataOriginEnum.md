# IfcDataOriginEnum

_IfcDataOriginEnum_ identifies the origin of time data.

> HISTORY&nbsp; New enumeration in IFC2x2.  
>

## Items

### MEASURED
The origin of the time data is a measurement device.

### SIMULATED

### NOTDEFINED
