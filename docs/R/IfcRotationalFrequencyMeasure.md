IfcRotationalFrequencyMeasure
=============================
_IfcRotationalFrequencyMeasure_ is a measure of the number of cycles that an
item revolves in unit time.  
  
Usually measured in cycles/s.  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
