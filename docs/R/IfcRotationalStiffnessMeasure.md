IfcRotationalStiffnessMeasure
=============================
_IfcRotationalStiffnessMeasure_ is a measure of rotational stiffness.  
  
Usually measured in Nm/rad.  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
