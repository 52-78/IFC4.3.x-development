IfcElectricConductanceMeasure
=============================
_IfcElectricConductanceMeasure_ is a measure of the electric conductance.  
  
Usually measured in Siemens (S, 1/Ohm = A/V).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
