IfcElectricCapacitanceMeasure
=============================
_IfcElectricCapacitanceMeasure_ is a measure of the electric capacitance.  
  
Usually measured in Farad (F, C/V = A s/V).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
