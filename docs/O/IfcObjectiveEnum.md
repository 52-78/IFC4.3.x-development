IfcObjectiveEnum
================
_IfcObjectiveEnum_ is an enumeration used to determine the objective for which
purpose the constraint needs to be satisfied.  
  
> HISTORY  New enumeration in IFC2.0.  
  
{ .change-ifc2x4}  
> IFC4 CHANGE  Extended to include CODEWAIVER.
