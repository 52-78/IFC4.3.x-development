IfcFaceBound
============
{ .extDef}  
> NOTE  Definition according to ISO/CD 10303-42:1992  
> A face bound is a loop which is intended to be used for bounding a face.  
  
> NOTE  Entity adapted from **face_bound** defined in ISO 10303-42.
