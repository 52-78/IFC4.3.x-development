IfcFrequencyMeasure
===================
_IfcFrequencyMeasure_ is a measure of the number of times that an item
vibrates in unit time.  
  
Usually measured in cycles/s or Hertz (Hz).  
  
Type: REAL  
  
> HISTORY  New type in IFC2.0.
