IfcLinearVelocityMeasure
========================
_IfcLinearVelocityMeasure_ is a measure of the velocity of a body measured in
terms of distance moved per unit time.  
  
Usually measured in m/s.  
  
Type: REAL  
  
> HISTORY  New type in IFC2.0.
