IfcLengthMeasure
================
An _IfcLengthMeasure_ is the value of a distance.  
  
Usually measured in millimeters (mm).  
  
Type: REAL  
  
> NOTE  Type adapted from **length_measure** defined in ISO 10303-41.  
  
> HISTORY  New type in IFC1.5.1.
