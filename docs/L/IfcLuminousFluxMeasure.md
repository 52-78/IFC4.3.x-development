IfcLuminousFluxMeasure
======================
_IfcLuminousFluxMeasure_ is a measure of the luminous flux.  
  
Usually measured in Lumen (lm, Candela Steradian).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
