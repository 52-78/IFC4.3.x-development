IfcMagneticFluxMeasure
======================
_IfcMagneticFluxMeasure_ is a measure of the magnetic flux.  
  
Usually measured in Weber (Wb, V s).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
