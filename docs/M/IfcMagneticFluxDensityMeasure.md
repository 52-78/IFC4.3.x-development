IfcMagneticFluxDensityMeasure
=============================
_IfcMagneticFluxDensityMeasure_ is a measure of the magnetic flux density.  
  
Usually measured in Tesla (T, Weber/m2 = V s/m2).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
