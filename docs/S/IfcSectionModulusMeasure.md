IfcSectionModulusMeasure
========================
_IfcSectionModulusMeasure_ is a measure for the resistance of a cross section
against bending or torsional moment. It is usually measured in m\\\^3.  
  
Type: REAL  
  
> HISTORY  New type in IFC2x2.
