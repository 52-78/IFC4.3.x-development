IfcAreaDensityMeasure
=====================
_IfcAreaDensityMeasure_ is a measure of the density of a two-dimensional
object and is calculated as the mass per unit area.  
  
Usually measured in kg/m2.  
  
Type: REAL  
  
> HISTORY  New type in IFC4.
