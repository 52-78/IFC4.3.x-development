IfcAbsorbedDoseMeasure
======================
_IfcAbsorbedDoseMeasure_ is a measure of the absorbed radioactivity dose.  
  
Usually measured in Gray (Gy, J/kg).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
