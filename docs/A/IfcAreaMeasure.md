IfcAreaMeasure
==============
An area measure is the value of the extent of a surface.  
  
Usually measured in square metre (m2).  
  
Type: REAL  
  
> NOTE  Type adapted from **area_measure** defined in ISO 10303-41.  
  
> HISTORY  New type in IFC1.5.1.
