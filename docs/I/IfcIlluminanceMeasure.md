IfcIlluminanceMeasure
=====================
_IfcIlluminanceMeasure_ is a measure of the illuminance.  
  
Usually measured in Lux (lx, Lumen/m2 = Candela Steradian/m2).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
