IfcContextDependentMeasure
==========================
The value of a physical quantity as defined within the exchange context.  
  
Type: REAL  
  
> NOTE  Type adapted from **context_dependent_measure** defined in ISO
> 10303-41.  
  
> HISTORY  New type in IFC1.5.1.
