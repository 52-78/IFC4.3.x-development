IfcVaporPermeabilityMeasure
===========================
_IfcVaporPermeabilityMeasure_ is a measure of vapor permeability.  
  
Usually measured in kg / s m Pascal.  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
