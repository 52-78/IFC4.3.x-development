IfcTextDecoration
=================
The _IfcTextDecoration_ describes decorations that are added to the text of an
element. Values are:  
  
* none  
* underline  
* overline  
* line-through  
  
> NOTE  Type adopted from **text-decoration** defined in
> [CSS-1](../../../bibliography.htm#CSS1).  
  
> HISTORY  New type in IFC2x3.
