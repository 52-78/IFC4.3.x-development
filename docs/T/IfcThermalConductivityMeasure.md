IfcThermalConductivityMeasure
=============================
_IfcThermalConductivityMeasure_ is a measure of thermal conductivity.  
  
Usually measured in Watt / m Kelvin.  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
