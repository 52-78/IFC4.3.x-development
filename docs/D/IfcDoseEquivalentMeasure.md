IfcDoseEquivalentMeasure
========================
_IfcDoseEquivalentMeasure_ is a measure of the radioactive dose equivalent.  
  
Usually measured in Sievert (Sv, J/kg).  
  
Type: REAL  
  
> HISTORY  New type in IFC2x.
